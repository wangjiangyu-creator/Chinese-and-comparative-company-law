# Chinese and Comparative Company Law Portal

A Chinese-first static research portal for Chinese company law, authoritative cases, scholarship and practitioner commentary links, and comparative company law materials.

## Local Preview

```bash
npm install
npm run dev
```

## Build

```bash
npm run check
npm run build
```

## GitHub Pages

The repository includes `.github/workflows/deploy.yml`. After the project is pushed to GitHub, enable GitHub Pages with **Source: GitHub Actions** in the repository settings. The workflow builds the Astro site and deploys the generated `dist` folder.
